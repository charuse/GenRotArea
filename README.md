# GenRotArea
### Rotate Objects Via Linked Generator IDs
  
|Var|Desc|
|-----|-------|
|Obj_arg0|Rotate X|
|Obj_arg1|Rotate Y|
|Obj_arg2|Rotate Z|
  
*Value is added to current rotation value for all LiveActors with linked Generator IDs*  
*Only works for LiveActors*  
*Rotates only once, triggered when player first enters the AreaObj*
